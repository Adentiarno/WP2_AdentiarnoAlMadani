HTML

HTML adalah bahasa markup yang digunakan untuk menentukan tata letak dan elemen-elemen di dalam halaman web, seperti judul, paragraf, hyperlink, dan lain-lain. HTML berfungsi untuk membangun struktur website, membuat halaman website yang menarik, dan memudahkan pengguna untuk mengakses informasi melalui internet

Bahasa C

Bahasa C adalah bahasa pemrograman umum yang dapat digunakan untuk membuat berbagai aplikasi, termasuk sistem operasi, database, dan aplikasi lainnya. Bahasa C banyak menginspirasi bahasa pemrograman lain, seperti C++, Java, JavaScript, dan PHP. Dengan mempelajari bahasa C, Anda akan lebih mudah memahami dan menggunakan bahasa pemrograman lain yang turunan dari bahasa C.

Java

Java adalah bahasa pemrograman multiplatform yang berorientasi objek, dikembangkan oleh James Gosling pada tahun 1995. Java dapat digunakan untuk mengembangkan aplikasi mobile, desktop, dan website.
